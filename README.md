# parallels

Parallel callbacks made easy.

[![Build Status](https://travis-ci.org/cshum/parallels.svg?branch=master)](https://travis-ci.org/cshum/parallels)

```bash
npm install parallels
```

#### `var p = parallels()`
#### `var cb = p.push()`
#### `p.all(done)`
#### `p.race(done)`
#### `p.any(done)`
#### `p.some(count, done)`
